# 通知撰写规范

## 核心规则

1. **标题格式**：通知标题不能以"通知"二字开头。必须冠以部门前缀，格式为"{部门}通知"。
   - 例如：采购部通知、宣传部通知、销售部通知
   - 如果用户没有告知所在部门，使用"XX部通知"代替

2. **正文结构**：
   - 第一段：说明通知的背景和目的
   - 主体部分：列出具体事项，使用序号（一、二、三…）分条说明
   - 最后一段：提出落实要求或注意事项

3. **落款格式**：
   - 右下角写明部门名称（如"XX部"）
   - 注明发文日期，格式为"2026年X月X日"

4. **语言要求**：
   - 使用正式、规范的公文语言
   - 避免口语化表达
   - 信息完整、准确、无歧义

## 示例

### 示例1（未知部门）

```
XX部通知

全体员工：

根据国家法定节假日安排，结合我单位实际情况，现将2026年五一劳动节放假安排通知如下：

一、放假时间
2026年5月1日（周五）至5月5日（周二），共放假5天。5月6日（周三）正常上班。

二、值班安排
各部门根据工作需要安排值班人员，值班表于4月28日前报办公室。

三、注意事项
请各科室在放假前检查门窗、电源，做好安全防范工作。

祝大家节日愉快！

                                XX部
                                2026年4月25日
```

### 示例2（已知部门）

```
销售部通知

全体销售人员：

根据国家法定节假日安排，结合销售部实际情况，现将2026年五一劳动节放假安排通知如下：

一、放假时间
2026年5月1日（周五）至5月5日（周二），共放假5天。5月6日（周三）正常上班。

二、客户对接
请各销售小组在放假前与客户确认订单交付时间，避免节后延误。

三、注意事项
放假期间保持通讯畅通，遇紧急事务及时处理。

                               销售部
                                2026年4月25日
```
